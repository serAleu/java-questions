*Паттерны проектирования*

Существует 3 типа шаблонов проектирования:
    1) Creational (Порождающие) - удобное и безопасное создание объектов
или семейств объектов.
    2) Structural (Структурные) - построение удобных в поддержке иерархий
классов.
    3) Behavioral (Поведенческие) - эффективное и безопасное взаимодействие
между объектами, уменьшение связей.
