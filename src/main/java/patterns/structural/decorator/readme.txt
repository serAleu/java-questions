*Паттерн Декоратор*

1) Декоратор — это структурный паттерн проектирования, который позволяет
динамически добавлять объектам новую функциональность, оборачивая их в
класс - "обёртки".

2) Декоратор можно назвать "оберткой", так более точно передается его
суть.

3) Все "декорируемые" объекты имеют общий интерфейс.
