1) Java позволяет создавать одни классы внутри других, такие классы называются вложенными

2) Вложенные классы бывают статическими, не статическими, анонимными и локальными в методе

3) Чтобы понять, зачем нужны вложенные классы вспомни пример с велосипедом (внешний класс)
и рулем к нему (вложенный класс) и пример с самолетом (внешний класс) и чертеж к нему
(статические вложенный класс).

4) Понимание работы внутренних классов, а особенно анонимных вложенных классов, необходимо
для понимания как работает лямбда в Java.
