1) Объект внутреннего класса не может существовать без объекта внешнего класса.

2) У объекта внутреннего класса есть доступ ко всем переменным внешнего класса,
в том числе и к приватным.

3) Объект внутреннего класса нельзя создать в статическом методе внешнего класса
потому что когда создается объект внутреннего класса в него неявно передается
ссылка на объект внешнего класса (в статическом методе объекта внешнего класса
вообще может не существовать).

4) Внутренний класс не может содержать статические переменные и методы (додумайся
сам почему так).

5) При создании объекта внутреннего класса важную роль играет его модификатор
доступа, это влияет на то где в нашей программе мы сможем создавать экземпляры
внутреннего класса. Модификаторы доступа для внутреннего класса работают так же
как и для переменных.
