*Статические вложенные классы*

1) Статический вложенный класс может быть объявлен внутри обычного класса (НЕ в теле метода)

2) Объект статического вложенного класса НЕ хранит ссылку на экземпляр внешнего класса. В
этом его основное отличие от не статического вложенного класса. Вспоминай пример с чертежом
самолета.

3) Объект статического вложенного класса может существовать сам по себе, но при создании
все равно нужно указывать имя внешнего класса через точку.

4) Статический вложенный класс может обращаться только к статическим полям внешнего класса.
Это происходит потому, что не статические переменные должны принадлежать объекту класса, а
при использовании статического вложенного класса такой объект может отсутствовать. Тоже
самое касается и методов.
