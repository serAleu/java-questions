*Наследование в java*

1) Один класс может extends другой класс, таким образом он может переиспользовать код родительского класса.
2) Наследуется все кроме приватных переменных и методов.
3) Множественного наследования в java нет.
4) Если нет ключевого слово extends, то класс наследуется от Object.
5) Наследование позволяет одному классу приобретать свойства и атрибуты другого, а полиморфизм использует
унаследованные свойства для выполнения различных задач.
