*Ключевое слово var*

1) Начиная с java10 в язык было добавлено ключевое слово var, которое
позволяет определять переменную.

2) Слово var становится вместо типа данных, а сам тип переменной выводится
из того значения, которое ей присваивается.

3) Если переменная объявлена с помощью var, то мы обязательно должны
инициализировать её сразу при объявлении переменной.

4) Если очень хочется, то var можно использовать вместе с дженериками, но
умные люди говорят, что лучше не надо.
