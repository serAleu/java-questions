*Инкапсуляция в java*

1) Основной принцип инкапсуляции - сокрытие реализации в данных методах.
2) Один из способов применения инкапсуляции в java является использование модификаторов доступа.
3) Доступ к приватным полям обеспечивается при помощи геттеров и сеттеров.
4) В качестве примера усиления инкапсуляции можно привести использование вложенных классов
(см. package nested_classes).
